**Use this for EVERY call - it ensures nothing critical is missed.**

## ğŸ“ TRAINING NOTES

### Common New Agent Mistakes:
1. **Skipping safety questions** - Always ask about danger FIRST
2. **Jumping to solutions** - Gather info first, then diagnose
3. **Under-triaging** - When in doubt, escalate severity UP
4. **Poor documentation** - Write it down NOW, not later
5. **Not setting expectations** - Tell customer what happens next

### How to Handle Difficult Situations:
- **Panicking customer** - Stay calm, be reassuring, speak slowly
- **Angry customer** - Acknowledge, apologize, focus on solution
- **Technical customer** - Match their expertise level, skip basics
- **Non-technical customer** - Use analogies, avoid jargon
- **Language barrier** - Speak slowly, use simple words, confirm understanding

### When to Break from the Checklist:
- **Obvious emergency** - Skip straight to Emergency Protocol
- **Repeat caller** - May skip some identification questions
- **Very technical crew** - Can abbreviate intake if they provide detailed info
- **Multiple simultaneous issues** - Focus on most severe first
